
# Details
Type: The type of fixture, such as a lamp post, an altar, etc.
Name: The officially designated name of the fixture.
Common Names: [Optional] Names and titles the fixture may be known by other than its official name.
Tags: Simple tags that help identify the nature of the fixture, such as "Light" and "Supernatural".
Perceived Value: A short description that indicates how inhabitants of the setting would value the fixture.

# Appearance
A description of the fixture's appearance and condition. This can also include the fixture's original appearance, and how it has changed over time.

# Mechanics
Contains information about the fixture's function and mechanics, such as relating to storage, security measures against unwanted intrusion, etc. This section should describe what is unique about this fixture, how it is meant to be used, and any important or special effects it has. If the fixture is magical, this section should include details about the nature of that magic, how the magic works, the limitations of its magic, etc.

# Plot Significance
Contains information about how this fixture is important, unique, and/or relevant to the setting, including details about how it affects those who use it, encounter it, etc. This section should describe conflicts that may result from the introduction, presence, or use of this fixture.

# History
Contains information about the significant and impactful moments of this fixture throughout history, including significant uses by its owners, transformations it has undergone, etc. It may include details about it's creator and the creation process if it is a particularly important story or if the fixture is particularly powerful or renown.
