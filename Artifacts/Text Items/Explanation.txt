
# Details
Type: The type of text item, such as a book, a document, a tome, a dictionary, a note, a movie script, etc.
Name: The officially designated name of the text item.
Common Names: [Optional] Names and titles the text item may be known by other than its official name.
Tags: Simple tags that help identify the nature of the text item, such as "Law" and "Magic".
Perceived Value: A short description that indicates how inhabitants of the setting would value the text item.

# Appearance
A description of the text item's appearance and condition. This can also include the text item's original appearance, and how it has changed over time.

# Mechanics
Contains information about the text item's function and mechanics, how it is meant to be used, and any important or special effects it has. If the text item is magical, this section should include details about the nature of that magic, how the magic works, the limitations of its magic, etc.

# Plot Significance
Contains information about how this text item is important and relevant to the setting, including details about how it affects those who use it, encounter it, etc. This section should describe conflicts that may result from the introduction, presence, or use of this text item.

# History
Contains information about the significant and impactful moments of this text item throughout history, including significant uses by its owners, transformations it has undergone, etc. It may include details about it's creator and the creation process if it is a particularly important story or if the text item is particularly powerful or renown.
